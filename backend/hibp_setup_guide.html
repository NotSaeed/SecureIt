Have I Been Pwned API Setup

To enable real-time breach checking with the official Have I Been Pwned database, you need to configure an API key.

Important: Without an API key, the system will use demo mode with simulated data for testing purposes.

Setup Instructions

Step 1: Get an API Key

Visit haveibeenpwned.com/API/Key to purchase an API subscription.

Pricing: API access requires a subscription starting from $3.50/month for basic usage.

Step 2: Configure the API Key

Edit the file: backend/config/hibp_config.php

Find this line:

'API_KEY' => 'YOUR_HIBP_API_KEY_HERE',

Replace YOUR_HIBP_API_KEY_HERE with your actual API key:

'API_KEY' => 'your-actual-api-key-here',

Step 3: Enable Real API Mode

In the same file, find this line:

'DEMO_MODE' => true,

Change it to:

'DEMO_MODE' => false,

Step 4: Adjust Rate Limits (Optional)

Based on your subscription plan, you may need to adjust the rate limit:

'RATE_LIMIT_RPM' => 60, // Adjust based on your plan

Subscription Plans:

  • Pwned 1: 10 RPM
  • Pwned 2: 60 RPM
  • Pwned 3: 200 RPM
  • Pwned 4: 1,000 RPM

Step 5: Test the Configuration

Return to the Security Center and open the "Have I Been Pwned" tool. The configuration status should show "Real API: Connected".

Benefits of Real API:
  • Access to 800+ real data breaches
  • Up-to-date breach information
  • Paste monitoring capabilities
  • Accurate security assessments

Security Considerations

  • Never commit API keys to version control
  • Restrict file permissions on the config file
  • Monitor API usage to avoid rate limits
  • Keep your API key secure and regenerate if compromised

Demo Mode

If you prefer to test the system without purchasing an API key, demo mode provides:

  • Simulated breach data for testing
  • Sample breach responses
  • Full UI functionality
  • No API costs

Need Help?

If you encounter any issues during setup:

  • Check that the config file is writable
  • Verify your API key is correct
  • Ensure your server can make HTTPS requests
  • Check the browser console for error messages
