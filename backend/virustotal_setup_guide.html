Step 1: Create VirusTotal Account

Visit the VirusTotal website and create a free account to get your API key.

Create VirusTotal Account

Step 2: Get Your API Key

After creating your account:

  • Log in to your VirusTotal account
  • Go to your profile settings
  • Navigate to the "API Key" section
  • Copy your personal API key
Get API Key

Step 3: Configure SecureIt

Open the VirusTotal configuration file and update your API key:

backend/config/virustotal_config.php

Replace the placeholder with your actual API key:

'api_key' => 'your_actual_virustotal_api_key_here'

Step 4: API Rate Limits

Free Account Limits:
  • 4 requests per minute
  • 500 requests per day
  • 5,500 requests per month
  • File size limit: 32MB

The system automatically handles these limits and will show appropriate messages when limits are reached.
